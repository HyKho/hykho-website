Frequently Asked Questions

How do I log in to the app?

You can log in using one of two methods:

  • QR Code: Scan the QR code from your HyKho web admin interface. This is the recommended method.
  • Manual Token: Enter your API token manually if you have it.
The camera is not working for QR code scanning

Make sure you've granted camera permissions:

  1. Go to your device Settings
  2. Find "HyKho Mobile" in the app list
  3. Enable Camera permissions
  4. Return to the app and try scanning again
The app is not syncing data

Check the following:

  • Ensure you have an active internet connection
  • Check if your ERP server is accessible
  • The app will automatically sync when connection is restored
  • Try logging out and logging back in
I forgot my API token

You can generate a new QR code from your HyKho web admin interface. The QR code contains all necessary authentication information.

The app crashes or freezes

Try these troubleshooting steps:

  1. Close the app completely and reopen it
  2. Restart your device
  3. Update to the latest version of the app
  4. Clear app cache (Settings → Clear Cache)
  5. If the problem persists, contact support with details about when the crash occurs
How do I change the language?

Go to Settings → Language and select your preferred language. The app supports English and Vietnamese.

Can I use the app offline?

Yes! HyKho Mobile supports offline mode. You can view and work with cached data even without internet. Data will sync automatically when you're back online.
